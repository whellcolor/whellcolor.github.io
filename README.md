## ⚡ Overview

WCC (Whellcolor Cube Crypto) is a decentralized Web3 dashboard ecosystem that combines:

- 🪙 ERC-20 Token System
- 💰 Staking & Reward Mechanism
- 🏦 Treasury / Vault System
- 🔌 Wallet Connection via Web3
- 📊 Modular DApp Interface (DEX / NFT / Miner / Cube)

This project is designed as a **fully on-chain + frontend hybrid Web3 application**.

## 🧠 Core Features

### 🔌 Wallet Integration
Supports wallet connection using modern Web3 SDK:
- MetaMask
- WalletConnect
- Injected wallets

Powered by:
- Reown AppKit

---

### 🪙 WCC Token System
- ERC-20 token standard
- Fixed max supply model
- Mint-controlled distribution
- Future DEX listing ready

---

### 💰 Staking System (Passive Rewards)
Users can:
- Stake WCC tokens
- Earn block-based rewards
- Claim rewards anytime
- Unstake liquidity

---

### 🏦 Treasury System
- Accepts ETH deposits
- Controlled withdrawal mechanism
- Used for ecosystem funding

---

### 🧩 Modules
- 📈 DEX Interface (UI Layer)
- 🖼 NFT Cube System (planned mint module)
- ⛏ Miner System (staking simulation / reward engine)
- 🧊 Cube System (interactive Web3 UI layer)
